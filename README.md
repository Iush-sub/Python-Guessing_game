# Guess Game

A simple command-line number guessing game built with Python.

## Description

Guess Game is a fun CLI application where the player tries to guess a randomly generated number. The game awards points based on how many attempts are needed to find the correct answer.

## Features

* Random number generation
* Multiple levels
* User-friendly command-line interface
* Replay option

## Technologies Used

* Python 3
* Random module

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Iush-sub/guess_game.git
```

2. Navigate to the project folder:

```bash
cd guess_game
```

3. Run the program:

```bash
python main.py
```

## How to Play

1. Start the game.
2. Choose your level.
3. Enter a guess when prompted.
4. The game checks your answer.
